# Implementation of a Cloud-Native Threat Intelligence Platform for Real-Time Correlation of Security Events

The rapid growth of cyber threats and the increasing complexity of modern IT infrastructures require organizations to adopt advanced security monitoring and threat detection mechanisms. Traditional security solutions often struggle to process large volumes of heterogeneous security data in real time, resulting in delayed incident detection and response. To address these challenges, this diploma project focuses on the implementation of a cloud-native threat intelligence platform designed for real-time correlation of security events.

The proposed platform integrates threat intelligence feeds from external sources such as AbuseIPDB and AlienVault OTX with security telemetry collected from infrastructure components and endpoints. By leveraging cloud-native technologies, the system provides scalable, resilient, and efficient processing of security events. Threat indicators, including malicious IP addresses, domains, URLs, and file hashes, are continuously synchronized and stored in a centralized repository.

A key feature of the platform is its ability to perform real-time correlation between incoming security events and known indicators of compromise (IOCs). The system automatically enriches collected logs with contextual threat intelligence information and applies detection rules to identify suspicious activities such as port scanning, brute-force attacks, privilege escalation attempts, reverse shell connections, and communications with known malicious infrastructure. When a match is detected, security alerts are generated and presented through a centralized monitoring dashboard.

The developed solution follows a cloud-native architecture based on containerized microservices and modern data-processing technologies. It incorporates components for log collection, event normalization, threat intelligence management, rule-based detection, alert generation, and visualization. The platform provides security analysts with enhanced situational awareness and reduces the time required to identify and investigate security incidents.

The practical outcome of this work is a functional Security Information and Event Management (SIEM) prototype capable of aggregating threat intelligence, correlating security events in real time, and supporting incident detection and response activities. The implementation demonstrates how cloud-native technologies can improve the scalability, flexibility, and effectiveness of modern cybersecurity operations.
